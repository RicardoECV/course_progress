Data do exercicio: 10/04/2024

1 - O que é HTML?

HTML significa "HyperText Markup Language" e é uma linguagem de marcação utilizada para a criação de páginas web.

2 - O que é um elemento HTML? Dê exemplos.

Um elemento HTML é o que vai indicar ao navegador como é mostrado o conteudo no site. Por exemplo o elemento <h1> é utilizado para mostrar um titulo.

3 - Que ferramentas são necessárias para construir um site HTML?

Nos dias de hoje um simples editor de texto é o sufeciente para construir um site HTML, mas o mais correto será sempre utilizar uma ferramenta especializada como o vscode.

4 - Quais os elementos base necessários para a criação de uma página HTML?

Os elementos base necessários para a criação de uma página html são 4. O <DOCTYPE html>, <html>, <head> e <body>.

5 - O elemento "br" necessita de ser fechado?

Não. Porque o elemento <br> é um elemento vazio.

6 - Para que servem os atributos? Dê um exemplo.

Os atributos são utilizados para colocar informação adicional nos elementos. Como por exemplo o atributo <height> altera o tamanho de uma imagem no elemento <img>.
